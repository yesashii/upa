Para poder realizar un cambio de valor de dinero en la cuenta corriente, se debe ingresar al SGA

 

Y luego ingresamos al módulo de ANALIZAR CUENTA CORRIENTE ingresando el Rut del alumno.

 

 

 

Ingresaremos con el Rut 17847224-0 para realizar una muestra del proceso que se debe realizar.

 

 

Estos son los datos que enviaron por correo para que realizaremos la modificación de los valores.

Rut 17.847.224-0

1016 $108.333 (correcto 108.300)

1017 $108.333 (correcto 108.300)

1018 $108.334 (correcto 108.400)

 

Para poder buscar los datos en la Base de Datos, necesitamos es el N° Docto y el valor de estos.

Y corremos el siguiente Script, donde se trabajara con la tabla DETALLE_INGRESOS.

select ding_mdetalle,ding_mdocto,* from detalle_ingresos where ding_ndocto in (1016,1017,1018) and ding_mdocto in (108334,108333)

 

Estos son los resultados del Script.

 

 

Para poder realizar el cambio uno por uno, rescataremos los datos del campo INGR_NCORR.

 

 

UPDATE detalle_ingresos SET ding_mdetalle = '108300',ding_mdocto= '108300'  where ingr_ncorr =1126273

UPDATE detalle_ingresos SET ding_mdetalle = '108300',ding_mdocto= '108300'  where ingr_ncorr =1126274

UPDATE detalle_ingresos SET ding_mdetalle = '108400',ding_mdocto= '108400'  where ingr_ncorr =1126275

 

Esto es uno de las cuatro cosas que debemos realizar para el cambio de valor.

 

 

Segundo Script se trabajara con la tabla INGRESOS.


select ingr_mdocto, ingr_mtotal,* from ingresos where ingr_ncorr in (1126273,1126274,1126275)

 

Para poder realizar este cambio, necesitamos el INGR_NCORR anteriormente ocupado.

 

UPDATE ingresos SET ingr_mdocto = '108300',ingr_mtotal= '108300' where ingr_ncorr = 1126273

UPDATE ingresos SET ingr_mdocto = '108300',ingr_mtotal= '108300' where ingr_ncorr = 1126274

UPDATE ingresos SET ingr_mdocto = '108400',ingr_mtotal= '108400'  where ingr_ncorr = 1126275

 

El tercer Script se trabajara con la tabla ABONOS.


select abon_mabono,* from abonos where ingr_ncorr in(1126273,1126274,1126275)

 

Y estos son los Script que debemos ejecutar para realizar el cambio.

UPDATE abonos SET abon_mabono = '108300' where ingr_ncorr =1126273

UPDATE abonos SET abon_mabono = '108300' where ingr_ncorr =1126274

UPDATE abonos SET abon_mabono = '108400' where ingr_ncorr =1126275

Y para realizar el último cambio para que se vea reflejado el cambio de los valores, se ejecuta el Script.


select dcom_mcompromiso,* from detalle_compromisos where comp_ndocto=137195 and tcom_ccod=1 and dcom_ncompromiso in (3)

 

Y se ejecuta el siguiente Script por cada compromiso.

UPDATE detalle_compromisos SET dcom_mcompromiso = '108300' where comp_ndocto=137195 and tcom_ccod=1 and dcom_ncompromiso in (1)

UPDATE detalle_compromisos SET dcom_mcompromiso = '108300' where comp_ndocto=137195 and tcom_ccod=1 and dcom_ncompromiso in (2)

UPDATE detalle_compromisos SET dcom_mcompromiso = '108400' where comp_ndocto=137195 and tcom_ccod=1 and dcom_ncompromiso in (3)

 

Y con esto refrescamos la página para ver el resultado del cambio.
