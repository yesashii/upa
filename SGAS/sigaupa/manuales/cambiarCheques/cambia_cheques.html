EN TESORERÍA

 

Primero se busca el cheque con DING_NDOCTO

select *

from   detalle_ingresos

where  ding_ndocto = [int numero de cheque]

       and ting_ccod = 3

       and ding_ncorrelativo = 1

 

Si tenemos un resultado entonces podemos actualizar el campo:

update detalle_ingresos

set    edin_ccod = 1

where  ding_ndocto = [int numero de cheque] 

       and ting_ccod = 3

       and ding_ncorrelativo = 1

 

Si se tiene más de un resultado hay que agregar más filtros como la fecha y montos.
